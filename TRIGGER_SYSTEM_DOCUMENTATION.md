## How to Use Triggers

### Basic Usage
Simply include a trigger keyword in your prompt:

```
Can you reason through this problem for me?
```

### Multiple Triggers
Combine triggers for more specific responses:

```
Please analyze and critique this proposal from a business perspective.
```

### In ChatArea
Use the Trigger Gallery in ChatArea to:
1. Click "Trigger Gallery" button
2. Search for specific triggers
3. Filter by category
4. Select triggers to apply
5. Send your message

### Advanced Features
- **Enable/Disable**: Toggle triggers on/off in the gallery
- **Custom Triggers**: Create custom triggers with specific instructions
- **Import/Export**: Share trigger configurations
- **Reset**: Return to built-in defaults

## System Prompt Integration

Triggers automatically:
1. Detect trigger keywords in user messages
2. Generate appropriate system prompts
3. Structure responses using XML-style tags
4. Preserve trigger-tagged content for display
5. Clean output while maintaining context

## Trigger Tag Format

Internally, triggers use XML-style tags:

```
<reason>
Step-by-step reasoning here
</reason>

<analyze>
Component analysis here
</analyze>
```

## Technical Details

### Storage
- Triggers stored in browser localStorage
- Custom triggers persist across sessions
- Built-in triggers always available
- Custom overrides take precedence

### Validation
- Tag names validated against VALID_TRIGGER_TAGS
- Duplicate prevention
- Category validation
- Instruction format checking

### Performance
- Efficient trigger detection using regex
- Lazy loading of trigger definitions
- Optimized category grouping
- Fast search and filter operations
