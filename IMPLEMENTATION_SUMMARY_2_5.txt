    - chat_collections (id, name, color, created_at, updated_at)

    - chat_tags (id, tag_name, color)
    - chat_tag_mapping (chat_id, tag_id)

Sharing Feature:
  Tables:
    - shared_chats (id, chat_id, share_token, is_public, password, expires_at)
    - message_comments (id, message_id, user_id, content, created_at)
    - comment_reactions (id, comment_id, emoji, user_id)

Bookmarks Feature:
  Tables:
    - bookmarks (id, message_id, content, folder_id, created_at)

    - user_analytics (id, user_id, model, tokens, created_at)

HOW TO USE FEATURES
================================================================================

Collections:
  1. Click "More Tools" in sidebar
  2. Select "Collections"
  3. Click "Create New Collection"
  4. Name your collection and pick a color
  5. Add chats by dragging or tagging
  6. Filter and search as needed

Sharing:
  1. Open a chat
  2. Click "Share" button in input footer
  3. Configure sharing options (public/private, password, expiration)
  4. Click "Create Share Link"
  5. Copy and send link to others
  6. Add comments and reactions in Comments tab

Bookmarks:
  1. View a chat with assistant messages
  2. Click the bookmark icon (üìå) on messages you want to save
  3. Icon turns yellow when bookmarked
  4. Go to "More Tools ‚Üí Bookmarks" to view all
  5. Organize into folders, search, and export

Advanced Analytics:
  1. Click "More Tools ‚Üí Advanced Analytics"
  2. View different metric tabs (Activity, Models, Tokens, Productivity)

  4. Export analytics as JSON for analysis
  5. Track usage patterns over time
